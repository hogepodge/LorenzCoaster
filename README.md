LorenzCoaster
=============

First-person rendering of the Lorenz Attractor.

## Running — Desktop (Processing 4)

Open `LorenzCoaster.pde` in the [Processing 4 IDE](https://processing.org/) and press Play.

## Running — Web (p5.js)

Open `index.html` in a browser, or serve it with any static file server, e.g.:

```
npx serve .
```

The web version (`sketch.js`) uses [p5.js](https://p5js.org/) and runs in any modern browser with WebGL support.
